First Section

-Mounting XFS File System:

sudo mount -t xfs /dev/nvme0n1p1 /mnt/xfs_mount_point

-Creating File in the Mount Point:

sudo touch /mnt/xfs_mount_point/example_file.txt

-Checking File Existence in Mount Point:

Is /mnt/xfs_mount_point/example_file.txt

-Unmounting the File System:

sudo umount /mnt/xfs_mount_point

-Verifying Unmount:

df -h | grep /mnt/xfs_mount_point
